# AfDB ROE Portfolio Intelligence Dashboard

Regional Operations Envelope (ROE) Portfolio Analysis — ADF-11 to ADF-16.
197 projects | $14.4 billion committed | Public data from AfDB Projects Portal.

## Features
- Portfolio overview with cycle/sector charts
- At-risk monitor ($4.2B flagged)
- Disbursement velocity analysis
- Searchable/filterable project explorer
- AI-powered query interface (Claude API)

## Deploy to Vercel
1. Push this repo to GitHub
2. Connect to Vercel (vercel.com → New Project → Import from GitHub)
3. Vercel auto-detects Next.js and deploys

## AI Query Setup
For the AI tab to work in production, add `ANTHROPIC_API_KEY` as an environment variable in Vercel project settings.
